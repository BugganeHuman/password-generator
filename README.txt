Simple password generator
generates random text depending on what conditions are selected, when you click the "copy" button, the password is copied. 
index.html – main HTML structure.
style.css – styles for layout and design.
script.js – JavaScript logic for password generation and copy functionality.
How to Use : 
1 Open index.html in a browser.
2 Select the desired options (uppercase, lowercase, numbers, special characters).
3 Enter the length of the password.
4 Click Generate Password to create a new password.
5 Click Copy to copy the password to the clipboard.
